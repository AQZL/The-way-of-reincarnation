# 背景故事
这是一个充满魔法的世界，而我们的主人公死后正好转生到了这个世界。在这个充满温馨而危险的世界里面，我们的主人公应该如何应对呢？他（她）会如何击败层层对手，让自己的城市、种族发展壮大？又会怎样用你的聪明才智去拯救这个异世界呢？你将化身主人公去到这个世界里面探险，去拯救他们吧！
# 整合包介绍
作为一款开放式的整合包，里面有着丰富的动植物及生物群系，无论是生存还是探险，体验感都很丰富；同时还有着多条游玩方向供玩家自己选择，可以成为魔王，也可以成为一名打败魔王的英雄！
# 整合包配置要求
建议分配内存：6 ~ 12 GB<br />
整合包Java适配状态：<br />
Java 11：【作者使用中】无不良反应，如果您游玩中遇到一些问题或许可以使用这个Java哦（需要在启动器中手动选择）。<br />
Java 8_271：【作者使用中】无不良反应，但是有些模组需要Java11才能完美适配。如果出现问题请切换Java11进行游玩本整合包。<br />
Java 12~17：【未知，不详】虽然说能启动本整合包但是记得使用“FTB备份”模组每小时（默认：2小时自动备份一次）备份一次哦<br />
# 整合包特点
1.整合包有着精美的UI界面，给转生者们带来美好的游戏体验哦!
2.拥有FTBQ任务系统：它将引领新手转生者学会怎么游玩这个整合包，但不偏向于成就目标；
3.模组数量看起来多，但大多是一些冒险魔法类的模组，同时还有许多使用辅助模组为游玩增添乐趣；
4.整合包优化做的很好；
5.一些实用类的模组也为转生者们提供了不错的帮助。

# 常见F&Q
_如何前往下界？_<br />
答：没转生前可以和原版一样正常前往转生后需要在野外找到红色传送门进入。<br />
_如何前往末地？_<br />
答：集齐12颗眼，详细内容看模组[ER]末地：创世。<br />
_为什么我死一次少一颗心？_<br />
答：神秘遗物模组中，因为你佩戴了开局送你的戒指[七咒之戒]导致每一次死亡都会掉落灵魂水晶你得给他捡回来。<br />
_如何重新选定种族？_<br />
答：合成一个Skill Reroll Scroll拿着长按右键即可使用。<br />
_能录制视频发到各大视频平台吗？_<br />
答：能。<br />
_实在找不到下界堡垒怎么办？_<br />
答：下界中输入该指令或许会对你有帮助哦。<br />
```/locate mana-and-artifice:demon_fortress```
